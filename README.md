# Currency Converter (Конвертер валют)

**Автор:** [Ваше Имя и Фамилия]

## Описание
Настольное приложение на Python (tkinter) для конвертации валют с использованием API [ExchangeRate-API](https://exchangerate-api.com).

## Функционал
- Конвертация между популярными валютами (USD, EUR, RUB, и др.)
- Проверка ввода (только положительные числа)
- Сохранение истории конвертаций в файл `history.json`
- Отображение последних операций в таблице

## Как получить API-ключ
1. Перейдите на сайт [https://exchangerate-api.com](https://exchangerate-api.com)
2. Зарегистрируйтесь (бесплатный тариф позволяет делать 1500 запросов в месяц).
3. Скопируйте API Key из личного кабинета.
4. Вставьте ключ в переменную `API_KEY` в файле `converter.py`.

## Запуск
```bash
python converter.py
```

## Примеры использования
1. Введите "100" в поле суммы.
2. Выберите USD в поле "Из".
3. Выберите RUB в поле "В".
4. Нажмите "Конвертировать".
5. Результат появится под кнопкой, а запись добавится в таблицу ниже.
